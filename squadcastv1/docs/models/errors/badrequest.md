# BadRequest

Represents a CircleCI error response for a 400 status code.


## Supported Types

### `errors.ResponseBodyError1`

```typescript
const value: errors.ResponseBodyError1 = {
  meta: {
    statusCode: 571062,
    errorMessage: "<value>",
  },
};
```

### `errors.ResponseBodyError2`

```typescript
const value: errors.ResponseBodyError2 = {
  meta: {
    status: "<value>",
    errorMessage: "<value>",
  },
};
```
