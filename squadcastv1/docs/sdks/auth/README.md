# Auth

## Overview

### Available Operations

* [authGetAccessToken](#authgetaccesstoken) - Get Access Token

## authGetAccessToken

Get access token to make authenticated HTTP requests to the Squadcast API.
Send your refresh token (obtained from the Squadcast web application) in the
`X-Refresh-Token` header.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="Auth_getAccessToken" method="get" path="/oauth/access-token" -->
```typescript
import { SquadcastSDK } from "@solarwinds/squadcast-sdk-typescript";

const squadcastSDK = new SquadcastSDK();

async function run() {
  const result = await squadcastSDK.auth.authGetAccessToken({
    xRefreshToken: "<value>",
  });

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { SquadcastSDKCore } from "@solarwinds/squadcast-sdk-typescript/core.js";
import { authAuthGetAccessToken } from "@solarwinds/squadcast-sdk-typescript/funcs/authAuthGetAccessToken.js";

// Use `SquadcastSDKCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const squadcastSDK = new SquadcastSDKCore();

async function run() {
  const res = await authAuthGetAccessToken(squadcastSDK, {
    xRefreshToken: "<value>",
  });
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("authAuthGetAccessToken failed:", res.error);
  }
}

run();
```
